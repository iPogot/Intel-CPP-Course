## To build project:
#### cd ./tests
#### ./run.sh compile

## To clean :
#### cd ./tests
#### ./run.sh clean

## To run ETE test:
#### cd ./tests
#### ./run.sh rf <input> <output> <answear>
  
## To run with input from file:
#### cd ./tests
#### ./run.sh rs <input>

## To run gtest:
#### cd ./tests
#### ./run.sh gt
